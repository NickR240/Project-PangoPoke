# Project-PangoPoke
**Purpose: **
-Quick retrival of informtation on Pokemon from the website www.serebii.net.
-Quick Type matchup information retrival.

**Planned Information presented:**
-Pokemon Name
-Pokemon Pokedex Entry
-Pokemon Types
-Pokemon Type Weaknesses(Organized by Neutral, x2, and x4)
-Pokemon Abilities and Descriptions
-List of Base Stats(HP |	Attack |	Defense |	Sp. Attack | Sp. Defense |	Speed)

**Current Version(v0.1) Uses:**
Purely Text Based Window
Retrieve Pokemon Name
Retrieve Pokemon Types.

**GUI Library:**
**Qt** - core, widgets, network, GUI
